# .principles inspect context — a11y
# Machine-executable pre-scan patterns per principle
# Used by /audit Phase 5 (Pre-Scan) to narrow scope before LLM reasoning.
# Format: - `command` | SEVERITY_HINT | description

### A11Y-ALT-TEXT

- `grep -rnE '<img(?![^>]*\balt\s*=)[^>]*>' --include="*.html" --include="*.jsx" --include="*.tsx" --include="*.vue" --include="*.svelte" $TARGET` | HIGH | img element missing alt attribute
- `grep -rnE 'alt\s*=\s*["'"'"'](image|photo|picture|icon|logo|img)\b' -i --include="*.html" --include="*.jsx" --include="*.tsx" $TARGET` | MEDIUM | Generic or meaningless alt text

### A11Y-SEMANTIC-HTML

- `grep -rnE '<div[^>]*on[Cc]lick' --include="*.html" --include="*.jsx" --include="*.tsx" --include="*.vue" $TARGET` | HIGH | Clickable div — use button or anchor instead
- `grep -rnE '<span[^>]*on[Cc]lick' --include="*.html" --include="*.jsx" --include="*.tsx" $TARGET` | HIGH | Clickable span — use button instead
- `grep -rnE '<div[^>]*class\s*=\s*["'"'"'][^"'"'"']*\b(btn|button|link|nav-item)\b' --include="*.html" --include="*.jsx" --include="*.tsx" $TARGET` | MEDIUM | Div styled as interactive element

### A11Y-KEYBOARD-NAVIGATION

- `grep -rnE 'tabIndex\s*=\s*[{"]?-1[}"]?' --include="*.jsx" --include="*.tsx" --include="*.html" $TARGET` | MEDIUM | tabIndex -1 removes element from tab order — verify intentional
- `grep -rnE 'onMouseEnter|onMouseLeave|onMouseOver' --include="*.jsx" --include="*.tsx" $TARGET` | MEDIUM | Mouse-only hover handlers — verify onFocus/onBlur equivalents exist
- `grep -rnE 'role\s*=\s*["'"'"'](button|link|menuitem|tab|checkbox|radio)["'"'"']' --include="*.jsx" --include="*.tsx" --include="*.html" $TARGET` | HIGH | ARIA interactive role — verify tabIndex and keyboard handler present

### A11Y-COLOR-CONTRAST

- `grep -rnE 'color\s*:\s*#(?:aaa|bbb|ccc|999|888)' -i --include="*.css" --include="*.scss" --include="*.sass" $TARGET` | HIGH | Known low-contrast grey shorthand value
- `grep -rnE 'opacity\s*:\s*0\.[1-4][^0-9]' --include="*.css" --include="*.scss" $TARGET` | MEDIUM | Low opacity on text — verify resulting contrast ratio
