# .principles inspect context — code/cs
# Format: - `command` | SEVERITY_HINT | description
# $TARGET is substituted with the actual path at runtime.
# Git commands are valid — $TARGET is used as a git pathspec.

### CODE-CS-BOY-SCOUT — The Boy Scout Rule

- `git diff HEAD -- $TARGET | grep -nE '^\+.*(TODO|FIXME|HACK|XXX)'` | LOW | New or changed lines introducing TODO/FIXME/HACK markers in touched files
- `git diff HEAD -- $TARGET | grep -nE '^\+.*\b[0-9]{4,}\b'` | LOW | New or changed lines introducing large magic numbers in touched files
