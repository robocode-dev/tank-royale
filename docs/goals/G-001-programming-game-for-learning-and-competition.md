# G-001 — A programming game for learning and competition

**Who:** Flemming N. Larsen (maintainer) and the Robocode community — programmers who learn by making bots fight.

**What:** Robocode Tank Royale — a programming game where players code autonomous tank-bots that compete in a virtual battle arena, playable across platforms (Java/JVM, .NET, Python, TypeScript) over a WebSocket protocol. *"Build the best — destroy the rest!"*

**Why:** Help users develop programming and AI skills in a fast-running, real-time competitive game; serve education with clear goals and benchmarkable performance; enable competition (e-sports for programming games). Backward compatibility with bot behavior and game physics is part of the promise — bots people wrote keep working.

The repo's existence and its released distribution channels (GitHub Releases, Maven Central, NuGet, PyPI) are the acceptance. Extracted from `openspec/project.md` at CH-001.
