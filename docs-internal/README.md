# docs-internal

Internal engineering documentation for Robocode Tank Royale. This directory is not part of the public API or user-facing docs — it is intended for developers working on the platform itself.

This directory contains two top-level artifacts: [DEBUGGING-GUIDE.md](./DEBUGGING-GUIDE.md) covers how to diagnose timing, event, and protocol bugs, and [architecture/](./architecture/) holds all Architecture Decision Records, C4 views, message schemas, and business flow diagrams. See [INDEX.md](./INDEX.md) for the structured file listing.
